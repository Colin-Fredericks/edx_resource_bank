The situation of this problem is simulated in the following applet.

  • Position the file cabinet at the bottom of the incline using the position control in the bottom right menu.
  • Give an angle to the incline with the ramp angle control.
  • By increasing slowly the applied force you will see how the force of static friction changes its magnitude and direction to keep the cabinet at rest.

There is a limit value of the applied force for which the cabinet starts to move up. Do not forget to check the force vectors so you can see the changing free body diagram.
