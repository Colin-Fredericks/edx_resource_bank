Objects Constrained to Move with Equal Accelerations

In this module we will discuss several examples where two or more objects are interacting through constraint forces such as ropes and contact forces. The constraint forces will restrict the motion of the objects involved in the system so that they move together, almost as if they were a single object. Examples include two boxes, one at the top of the other, on an accelerating elevator as discussed in the previous module. In this situation the constraint forces are the normal forces between the boxes and the normal force between the lower box and the elevator's floor. As a result, the boxes are constrained to move with the same acceleration as the elevator. In this example and in the three examples we will see in this module, the relationship between the accelerations of the different objects in the problem is very simple: the accelerations are equal in magnitude and direction.

Learning Goals

After completing this module, you should be able to:

  • Recognize different types of constraints.
  • Solve problems where more than one object is constrained to move with the same acceleration.

Related Acceleration Constraint

The Related Accelerations Constraint identifies the relationship between the accelerations of two or more objects moving under constraint forces. We will see three types of relationships:

  • Accelerations with equal magnitude and equal direction (see examples in this module)
  • Accelerations with equal magnitude but different direction (next module: e.g. two objects connected via a string through a single pulley)
  • Accelerations with different magnitude and different directions (e.g several connected objects with more than one pulley, an object sliding down a moving wedge, etc.)

For the problems in this module and in the following modules, the "standard approach" when applying Newton's laws is:

  1. Isolate each body and treat it as a point particle.
  2. Identify the interactions between each object and other objects.
  3. Classify the resulting forces and draw free body diagrams for each object.
  4. Pick a convenient coordinate system for each object.
  5. Write down Newton's Second Law for vector components consistent with the coordinate system chosen in step 4.
  6. Express any new mathematical relations that exist due to motion constraints (Related Acceleration Constraint)  or Newton's 3rd law pairs
  7. Solve the equations from steps 5 and 6.

Note that in step 1. we start by selecting separate systems, one per object involved in the problem. Only in the case when all the objects are restricted to move with the same acceleration, as in the examples of this module, can we consider a multiple object system as a point particle of mass equal to the combined masses of the objects. If we start the solution of the problem by selecting the multi-object system then we do not need to go through step 6. In that case we are already using the fact that the accelerations are equal by treating all the objects as one; the Newton's 3rd law pairs are the internal forces which we ignore.
